# Rule-Based AI Chatbot

## Objective

A simple chatbot built using Python that responds to predefined user inputs.

## Features

- Handles greetings
- Responds to common questions
- Runs continuously
- Exit command support
- Uses Python dictionary lookup

## Technologies Used

- Python 3

## Run

```bash
python chatbot.py
```

## Example

You: hello

Bot: Hi there!

You: exit

Bot: Exiting chatbot...
